# Qualification Client

> Systeme complet pour qualifier un prospect AVANT d'investir du temps : scoring, archetype, secteur, negociation.

## When to Use

- Quand un nouveau prospect arrive
- Avant de rediger un devis
- Pour adapter la strategie de negociation
- Lors de la revue hebdomadaire du pipeline

## How It Works

### 1. Les 7 Archetypes Client

#### Dirigeant PME
```
PSYCHOLOGIE : Veut du concret, pas du blabla. Budget = investissement.
SIGNAUX : Parle chiffres, demande le ROI, decide vite.
RED FLAGS : "Faites-moi un prix", veut tout pour rien.
BUDGET TYPIQUE : 800-2000€
NEGOCIATION : Direct, transparent. Montrer la valeur multi-livrable.
TON : Direct, rassurant, concret.
EVITER : Jargon technique, options trop complexes.
```

#### Responsable Communication
```
PSYCHOLOGIE : Process, validation interne, besoin de justifier le budget.
SIGNAUX : Demande un brief, parle de "strategie", veut des KPI.
RED FLAGS : Chaine de validation trop longue (>3 niveaux).
BUDGET TYPIQUE : 1500-5000€
NEGOCIATION : Data-driven. ROI, KPI, benchmark concurrent.
TON : Professionnel, structure.
EVITER : Tutoiement premature, devis simplifie.
```

#### Agence Evenementiel
```
PSYCHOLOGIE : Pro entre pros. Fiabilite > creativite.
SIGNAUX : Brief structure, deadlines serrees, volume.
RED FLAGS : Tarifs imposes sans negociation, paiement > 60 jours.
BUDGET TYPIQUE : 700-3000€
NEGOCIATION : Volume, recurrence, partenariat long terme.
TON : Pro, efficace.
EVITER : Discours creatif (ils gerent eux-memes).
```

#### Startup Founder
```
PSYCHOLOGIE : Move fast. Budget serre mais besoin d'impact.
SIGNAUX : "On leve bientot", energie, vision.
RED FLAGS : "On paiera quand on aura leve", equity au lieu de cash.
BUDGET TYPIQUE : 600-2500€
NEGOCIATION : Rapidite, impact, premiere collab a prix juste.
TON : Decontracte mais pro.
EVITER : Process long, devis 3 pages.
```

#### Artisan Local
```
PSYCHOLOGIE : Peur du prix. Besoin de pedagogie.
SIGNAUX : "C'est combien ?", premiere fois en video, decouvre le process.
RED FLAGS : Compare avec le neveu qui a un iPhone.
BUDGET TYPIQUE : 400-900€
NEGOCIATION : Pedagogie, avant/apres, ROI Instagram simple.
TON : Chaleureux, pedagogique.
EVITER : Termes techniques, budgets eleves d'emblee.
```

#### Client Luxe
```
PSYCHOLOGIE : L'image est tout. Prix = qualite percue.
SIGNAUX : Parle "direction artistique", references haut de gamme.
RED FLAGS : Aucun (sauf si ne paie pas).
BUDGET TYPIQUE : 2000-10000€+
NEGOCIATION : Ne JAMAIS baisser le prix. Premium = premium.
TON : Elegant, soigne.
EVITER : Tarif decouverte, discounts.
```

#### Client Toxique
```
PSYCHOLOGIE : Veut tout, paie rien, change d'avis tout le temps.
SIGNAUX : Scope creep des le premier message, negocie avant de comprendre.
RED FLAGS : "Je connais quelqu'un qui fait ca pour 200€",
           revisions infinies, paiement "a la livraison" sans acompte.
ACTION : REFUSER poliment ou facturer 50% d'avance.
```

### 2. Matrice de Scoring (/100)

```
CRITERE             | POINTS | EVALUATION
--------------------|--------|------------------------------------------
Budget clair        |   /30  | 30=annonce, 20=fourchette, 10=flou, 0=aucun
Besoin defini       |   /20  | 20=brief, 10=idee vague, 0=sait pas
Decideur direct     |   /15  | 15=decideur, 10=intermediaire, 0=inconnu
Timing defini       |   /10  | 10=date, 5=trimestre, 0=un jour
Potentiel recurrence|   /10  | 10=mensuel, 5=annuel, 0=one-shot
Communication       |   /10  | 10=reactive, 5=lent, 0=fantome
--------------------|--------|------------------------------------------
SOUS-TOTAL          |  /95   |

RED FLAGS (malus -10 chacun) :
  [ ] Negocie avant de comprendre le travail
  [ ] Compare avec "mon neveu qui a un iPhone"
  [ ] Scope creep dans le premier message
  [ ] Paiement conditionnel ("si ca marche")
  [ ] Chaine de validation > 3 niveaux

SCORE FINAL : [X] / 100
```

### 3. Seuils de Decision

```
80-100 : IDEAL — Devis immediat, tarif standard ou premium
60-79  : BON — Devis apres cadrage, tarif standard
40-59  : RISQUE — Cadrer fortement, acompte 50%, perimetre serre
< 40   : REFUSER — Poliment decliner ou orienter vers un concurrent
```

### 4. Playbooks de Negociation

#### "C'est trop cher"
```
REPONSE :
1. Accuser reception : "Je comprends, c'est un investissement."
2. Decomposer la valeur : "Le tarif inclut [liste detaillee]."
3. Comparer : "Un resultat equivalent en agence coute [2-3x plus]."
4. Option B : "On peut ajuster le perimetre : [version allegee] a [prix reduit]."

JAMAIS : Baisser le prix sans reduire le perimetre.
```

#### "Un concurrent fait moins cher"
```
REPONSE :
1. Qualifier : "Qu'est-ce qui est inclus dans leur proposition ?"
2. Differencier : "Voici ce que j'inclus en plus : [liste]."
3. Confiance : "Je prefere vous livrer un travail dont on sera fier tous les deux."

JAMAIS : S'aligner sur un prix sans connaitre la proposition concurrente.
```

#### "Tarif decouverte ?"
```
CONDITIONS :
  - Score prospect > 60
  - Potentiel recurrence reel (pas juste une promesse)
  - Reduction max 15% sur le premier projet
  - Engagement verbal sur un 2e projet si satisfait

FORMULATION :
"Pour une premiere collaboration, je peux faire un effort de 15%.
Si on est contents tous les deux, on part sur un tarif recurrence."
```

#### Scope Creep
```
DETECTION : Le client ajoute des demandes apres le devis valide.
REPONSE :
1. "Tres bonne idee ! Ca n'etait pas dans le devis initial."
2. "Je peux l'ajouter pour [montant supplementaire]."
3. "Ou on le prevoit pour un prochain projet."

REGLE : Toute demande hors devis = avenant ou nouveau devis.
```

### 5. Signaux Comportementaux

Observer pendant les echanges :

```
SIGNAUX POSITIFS :
  + Repond vite (< 24h)
  + Pose des questions precises
  + Partage des references visuelles
  + Parle du projet au futur ("quand on fera...")
  + A deja un budget en tete

SIGNAUX NEGATIFS :
  - Repond apres 3+ jours
  - Questions vagues ("c'est combien en gros")
  - Ne donne pas de brief
  - Parle au conditionnel ("on pourrait peut-etre...")
  - Compare systematiquement avec d'autres
```

### 6. Fiches Secteur Paris

#### Restauration
```
BUDGET MOYEN : 600-1500€
BESOIN : Video ambiance, plats, equipe, interview patron
FREQUENCE : 2-4x par an (saisons, nouveau menu)
ARGUMENTS : Instagram est crucial, avant/apres photos vs video
PIEGE : Veulent tout filmer en 2h pendant le service
```

#### Immobilier
```
BUDGET MOYEN : 800-2500€
BESOIN : Visite virtuelle, drone, plans interieurs
FREQUENCE : Par bien (ponctuel), mais volume possible
ARGUMENTS : Vente plus rapide, differenciation agence
PIEGE : "Juste des photos avec un peu de video"
```

#### E-commerce
```
BUDGET MOYEN : 500-2000€
BESOIN : Packshot video, lifestyle, UGC style
FREQUENCE : Mensuel a hebdomadaire (contenu RS)
ARGUMENTS : Taux de conversion, engagement RS
PIEGE : Volume enorme pour petit budget
```

#### Bien-etre / Sante
```
BUDGET MOYEN : 600-1800€
BESOIN : Ambiance, temoignages, process
FREQUENCE : 2-3x par an
ARGUMENTS : Confiance client, humaniser la marque
PIEGE : Contraintes image (patients, nudite, medical)
```

#### Tech / SaaS
```
BUDGET MOYEN : 1000-3000€
BESOIN : Demo produit, culture video, temoignages
FREQUENCE : Mensuel (contenu marketing)
ARGUMENTS : Cycle de vente, recrutement, investor relations
PIEGE : Jargon technique incomprehensible a filmer
```

#### Evenementiel
```
BUDGET MOYEN : 700-3000€
BESOIN : Aftermovie, teaser, captation live
FREQUENCE : Par event (mais fidele si ca marche)
ARGUMENTS : Promo prochain event, sponsors
PIEGE : "On vous paiera en visibilite"
```

## Examples

### Exemple : Qualification rapide

```
Richard : "J'ai un resto dans le 11e qui m'a contacte sur Instagram"

QUALIFICATION RAPIDE :
  Archetype probable : Artisan Local ou Dirigeant PME
  Questions a poser :
  1. C'est le patron qui vous contacte ? (decideur)
  2. Il a un budget en tete ? (budget)
  3. Il veut quoi exactement ? (besoin)
  4. Pour quand ? (timing)

  En attendant les reponses :
  Score provisoire : ~50 (peu d'info)
  Strategie : Pedagogie + proposition claire
  Tarif prevu : 600-900€ (fourchette artisan/resto)
```
