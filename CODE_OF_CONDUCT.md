# Code of Conduct — Dhwani RIS

## Our Commitment

Dhwani Rural Information Systems is committed to providing a respectful, inclusive, and professional environment for all team members contributing to this repository. This code applies to all interactions within the repo — pull requests, code reviews, issues, and discussions.

## Expected Behavior

- Treat all team members with respect and professionalism
- Provide constructive, specific feedback in code reviews — critique the code, not the person
- Ask questions when something is unclear rather than making assumptions
- Share knowledge openly — skills, patterns, and domain expertise benefit everyone
- Take ownership of your contributions — test before you PR, fix what you break
- Respect the skill graph structure — don't duplicate, extend instead

## Unacceptable Behavior

- Dismissive or hostile responses to questions from any team member, regardless of seniority
- Committing credentials, tokens, API keys, or secrets to the repository
- Pushing directly to main without a PR and review
- Modifying proprietary domain skills without authorization
- Sharing repository content (especially proprietary domain skills) outside Dhwani RIS

## Enforcement

Violations should be reported to **Ankit Jangir** (ankitjangir.1690@gmail.com), who is the repository maintainer and responsible for enforcement.

- **First violation:** Private conversation and course correction
- **Repeated violations:** Temporary loss of push access
- **Serious violations** (credential leaks, unauthorized sharing of proprietary content): Immediate access revocation and escalation to management

## Scope

This code applies to all contributions to this repository, including code, documentation, reviews, and discussions. It extends to any context where you represent this project or use its content.
