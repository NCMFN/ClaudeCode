# Setup Guide

## Requirements
* Python 3.10+
* An OpenAI or Anthropic API key (or compatible LLM API endpoint)

## Installation
1. Clone the repository.
2. Navigate to the `multi_agent_framework` directory.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Set up environment variables for your chosen LLM (e.g., `OPENAI_API_KEY` or `ANTHROPIC_API_KEY`).

## Usage
Run the main script to start the multi-agent SDLC workflow:
```bash
python main.py
```
